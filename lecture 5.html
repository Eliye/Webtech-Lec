LESSON 5: HTTP Response Message

  • Status Line (CRLF terminated line consisting of three space-separated values)

      • HTTP Protocol version

    • Status Code

      • - a 3-digit integer result code to understand and satisfy request.

    • Reason Phrase

      • - descriptive representation of status code.

    • Response Header Fields

      • - allow the server to pass additional information about the response which cannot be placed in the Status- Line.

      Accept-Ranges

      allows the server to indicate its acceptance of range requests for a resource

      Age

      estimate the sender's amount of time since the response

      Etag

      provides the current value of the entity tag for the requested variant

      Location

      used to redirect the recipient to a location

      Proxy-Authenticate

      consists of a challenge that indicates the authentication scheme and parameters applicable to the proxy

      Retry-After

      indicate how long the service is expected to be unavailable

      Server

      contains information about the software used by the origin server to handle the request

      Vary

      indicates the set of request-header fields that fully determines, while the response is fresh

      WWW-Authenticate

      consists of at least one challenge that indicates the authentication schemes and parameters

  • Message Headers

    • - general, client request and/or entity headers

  • Empty Line

    • - a line with nothing preceding the CRLF

  • Message Body

    • - optional ; used to carry the entity-body associated with the request or response

  • Message Length

    • - message-body length that appears in the message

    Transfer-Encoding header field - is present and has any value other than "identity", then the transfer-length is defined by use of the "chunked" transfer-coding.

    Content-Length header field - is present, its decimal value in OCTETs represents both the entity-length and the transfer-length.

  • General Header Fields

    • - only applied to message that is being transmitted.
