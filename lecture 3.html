JavaScript (JS)

JS (Client Side Scripting)
  • Use dynamic typing
  • Not strongly type
  • like java and c++, one declaration
  • de facto language of the Web, imperative language
  • Standard ECMA - 262 : 7th edition (June 2016)
  • ECMA Script 5 - almost full support

Document Object Model - describes the structures of mark-up

Three ways to link a script
Externally Linked Script (Recommended practice)
    <head>
        <script> </script>
    </head>
Embedded Script
    <body>
        <script> </script>
    </body>
Inline Script
    <button onclick='console.log("inline script...");'>
        Click Me
    </button>
JS Attributes
Defer attribute
  • Fetch script, render blocking, script execute
  • Block execution and continue execution after all are being rendered.
    <script defer type = ‘text/javascript’ src = 'script.js'>
    </script>
async attribute
  • Asyncronous loading : fetch script, continue rendering, can execute
  • As soon as the script is available, it can already execute.
    <script async type = ‘text/javascript’ src = 'script.js'>
    </script>
