Lecture 1: HTML ELEMENTS

  • html - root element (parent element)
  • head - (child element)
  • title
  • base
  • link
  • meta - info about the author or version (authoring tool)
  • style - css
  • body - actual content of the document (child element)
  • article, aside, nav, section, header, footer (main html 5)
  • address, div
  • h1, h2, h3, h4, h5, h6 (headings)
  • p
  • hr - thematic break (documenting)
  • pre - preformatted text
  • blockquote - in line citations (larger block)
  • ol - ordered list, ul - unordered list
  • li
  • dl - description list
  • dt, dd - simple markup
  • figure - embedded function
  • figcaption - embedded function
  • a
  • em - emphasis
  • strong - bold
  • small - side comment
  • s - obsoleted text
  • cite - citation
  • q - quote
  • dfn - defining instance
  • abbr - subsequent usage
  • ruby, rb, rt, rte, rp - assign language annotation
  • data, time, embed- machine readable data
  • code, var
  • samp - sample output
  • kbd - sample input
  • sub - subscript, sup - superscript
  • i, b, u
  • bdi, bdo - mark bidirectional
  • span - generic divider like dive, embedded with block of text
  • br - line break
  • wbr - break opportunity (narrow)
  • ins - insertion, del - deletion
  • table
  • caption, colgroup, col, thread, tbody, tfoot
  • tr
  • th, ta
  • form
  • label, input, button, select
  • picture, img, iframe, embed, object, param, audio, video, source, track
  • map, area
  • svg - not html (scalable vector graphics)
  • canvas
  • script, noscript
  • template
