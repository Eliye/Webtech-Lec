Lecture 1: CSS History

  • CSS (Cascading Style Sheets)

    is a simple mechanism for adding style (e.g., fonts, colors, spacing) to Web documents.

  • Time Table
  • October 1994: Hakon Wium Lie released the first draft of “Cascading HTML Style Sheets”
  • May 1995: The discussions on the www-style mailing list
  • August 1996: Microsoft Internet Explorer became the first browser to support CSS.
  • December 1996: The CSS Level 1 W3C Recommendation was made
  • The release of CSS 1 supported:
      font properties
      text attributes
      alignment of text
      tables
      images and more
      colors of text and backgrounds
      spacing of words
      letters and lines
      margins
      borders
      padding and positioning
      and unique identification and classification of groups of attributes
  • May 1998: the W3C released CSS 2 which added new capabilities including z-index, media types, bidirectional text, absolute, relative and fixed positioning, and support for aural style sheets.
  • Between June 2011 and June 2012: the following four modules were released as formal recommendations: color, selectors level 3, namespaces and media queries.
