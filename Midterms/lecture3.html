Lecture 3: CSS Statements and CSS Rule Sets

CSS Statements

  • @ At - Rules
  • @ Charset
  • @ Import
  • @ Page
  • @ Namespace
  • @ Document
  • @ Font-face
  • @ Keyframes
  • @ Media

    • CSS - Rule Sets

    • Consist of a selector followed by a brace enclosed declaration block, which contains zero
      • P - selector
      • {} - declaration block
      • : - property separator
      • color - property name
      • green - property value
      • /* or */ - comment
      • // - invalid comment
