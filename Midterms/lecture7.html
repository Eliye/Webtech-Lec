Lecture 7: Client Side Scripting

Javascript - create a script then associate it with a webpage

  • Use dynamic typing
  • Not strongly type
  • like java and c++, one declaration
  • de facto language of the Web, imperative language

    • Document Object Model - describes the structures of mark-up

    • Standard ECMA - 262 : 7th edition (June 2016)
    • ECMA Script 5 - almost full support
    • Externally or Linked Script
      • < script type = 'text/javascript' src = 'script.js' > < / script >
      • console.log ('externally linked script...');
        • Attribute
        • Defer – do thing in parallel (wait until the document is rendered) < script defer type = ‘text/javascript’ src = ‘script.js’ > < / script >
        • Async – for asynchronous loading (don’t wait until it finish executing) < script async type = ‘text/javascript’ src=’script.js’ > < / script >
    • Embedded Script
    • < script > console.log('embedded script...'); function callMe(){ console.log ("you called me..."); < / script > }
    • Inline Script
      • < button onclick = 'console.log("inline script...");'> Click Me < / button>
