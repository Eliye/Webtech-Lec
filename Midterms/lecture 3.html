Lecture 4: CSS Selector Syntax

  • Selector
    • Structure used as a condition in a CSS rule to determine which elements in the document tree are matched by the selector and are thus targeted by the formatting specified in the CSS rule declaration block. The matched elements are called the subjects of the selector.
  • Selector-syntax
    • Chain of one or more sequences of simple selectors separated by combinators with one pseudo-element possibly appended to the last sequence.
  • Sequence of Simple Selectors
    • Chain of simple selectors not separated by combinators
    • Always starts with a type selector or a universal selector
    • Cannot contain other type selectors or universal selectors
  • Group of Selectors
    • Comma-separated list of selectors representing the union of all elements selected by each of the selectors in the list
      • Simple Selectors
        • Type selector - type of targeting elements
        • Universal selector - represented by an asterisk (*)
        • Attribute selector
          • [attr]
          • [attr = value]
          • [attr == value]
          • [attrl = value]
          • [attr^ = value]
          • [attr$ = value]
          • [attr* = value]
      • Class Selector - represented by a period
      • ID Selector - pound sign
      • Pseudo-class
        • Implicit class that is associated with the inherited classes or mark up
          • Dynamic pseudo class
            • link psuedo-classes
              • :link
              • :visited
          • User action pseudo-classes
            • :hover - roll over effect
            • :active - activate a particular element by clicking a button
            • :focus - form many input element
          • Target pseudo-class
            • :target
          • Language pseudo-class
            • :lang()
          • UI element states pseudo-classes
            • :enabled
            • :disabled
            • :checked
            • :indeterminate
          • Structural pseudo-classes
            • :root
            • :first-child
            • :last-child
            • :only-child
            • :nth-child()
            • :nth-last-child
            • :first-of-type
            • :last-of-type
            • :only-of-type
            • :nth-of-type()
            • :nth-last-of-type()
            • :empty
          • Negation pseudo-class
            • :not() - negates the logic of a selector
          • Combinators
            • descendant combinator (whitespace, i.e space, tab, linefeed, carriage return, form feed)
              • child combinator (>) direct/immediate descendant
              • sibling combinators (same parent)
                • adjacent sibling combinator (+)
                • general sibling combinator (~)
          • Pseudo-elements
            • ::first-letter - :first-letter
            • ::first-line - :first-line
            • ::before - :before
            • ::after - :after
