Lecture 6: CSS Declarations

Properties

  • Shorthand properties (e.g background,font) and longhand properties (background)
  • Vendor specific extensions (a.ka. vendor prefixes)
  • Experimental properties
    • -webkit-transform-origin;
    • -moz-tranzform-origin; (Mozilla-ghetto)
    • -ms-transform-origin;(Edge)
    • -o-transform-transform-origin;(Opera-presto)
  • Global property keywords
    • :inherit
    • :initial
    • :unset
  • Values
    • Keywords
    • Numbers
    • Dimensions (numbers with units, length, angle, duration
    • Length Units
      • Font-relative - em, ex, ch, rem (root element)
      • Viewport-percentage- vw, vh, wmin, vmax
      • Absolute-lengths - cm, mm, q, in, pt, pc
      • Angle Units - deg, grad, rad, turn
      • Duration Units - s, ms (animation, transition)
      • Frequency Units - hz, khz
      • Resolution Units - dpi, apem, appy (media queries)
    • Percentages - number with a percent sign (%)
    • URL's and URI's - URI function indicate resources that links
    • Colors - counters(), linear-gradient(), radial-gradient(), translate(), scale(), rotate()
    • rgb, #, hsl (hue, saturation, light), rgba, hsla
    • Functions - calc(), attr(), counter(), counters(), linear-gradient(), radial-gradient(), translate(), scale(), rotate(), etc.
