TOPIC 8: HTTP Status Codes

Informational (1XX)

100 Continue

101 Switching Protocol

    Success (2XX)

    200 Ok

    201 Created

    202 Accepted

    203 Non-Authoritative Information

    Certain part of response not coming from origin server.

    204 No Content

    Successful delete.

    205 Reset Content

    Instruct client can reset.

    206 Partial Content

    Server tells that this is not the whole resource.

      Redirection (3XX)

      300 Multiple Choice

      Respond with 300 (redirect) and gives choices.

      301 Moved Permanently

      302 Found

      303 See Other

      304 Not Modified

      305 Use Proxy

      Gives the URL of proxy using the location.

      306 (unused)

      307 Temporary Redirect

        Client Error (4XX)

        400 Bad Request

        401 Unauthorized

        402 Payment Required

        e.g. Online payments

        403 Forbidden

        Supposed not to access source.

        404 Not Found

        405 Method Not Allowed

        406 Not Acceptable

        407 Proxy Authentication Required

        408 Request Time-out

        409 Conflict

        Whatever request can not be done because of conflict.

        410 Gone

        Resource not longer there.

        411 Length Required

        Server expecting content length.

        412 Precondition Failed

        Condition has failed, con not proceed.

        413 Request Entity Too Large

        Denial of server.

        414 Request URI Too Large

        To long to be handled by server.

        415 Unsupported Media Type

        Can reject upload.

        416 Requested Range Not Satisfiable

        417 Expectation Failed

        426 Upgrade Required (new)

        Using feature that needs to be upgrade.

        Server Error (5XX)

        500 Internal Server Error

        501 Not Implemented

        502 Bad Gateway

        Gateway having problems.

        503 Service Unavailable

        Doing maintenance.

        504 Gateway Time-out

        505 HTTP Version Not Supported
