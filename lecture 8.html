HTTP Status Codes

Informational (1xx)

This class of status code indicates a provisional response, consisting only of the Status-Line and optional headers, and is terminated by an empty line.

From https://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html

100 Continue

This indicates that the client should proceed with the remaining part of the request. Learn More

101 Switching Protocol

This indicates that the client requested to switch protocol and server accepts the request. Learn More

102 Processing (WebDAV)

This informs the client that the server accepted the request but the request is not completed yet. Learn More

Success (2xx)

This class of status code indicates that the client's request was successfully received, understood, and accepted.

From https://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html

200 OK

The status code is used to indicate that the request was successful. Learn More

201 Created

The request has been completed and a new resource is created. Learn More

202 Accepted

The request has been accepted but has not been completed. Learn More

203 Non-Authoritative Information

Certain part of the response are not coming from the origin server. Learn More

204 No Content

The request was completed but no content needed to be return. Learn More

205 Reset Content

The request was completed and client should reset the document view. Learn More

206 Partial Content

Server tells that this is not the whole resource. Learn More

207 Multi-Status (WebDAV)

This provides multiple status for independent operations. Learn More

208 Already Reported (WebDAV)

This indicates avoiding enumeration of internal members of multiple bindings repeatedly. Learn More

226 IM Used

The GET request for the resource is completed and the response is a representation of the result of one or more instance-manipulations applied to the current instance. Learn More

Redirection (3xx)

This class of status code indicates that further action needs to be taken by the user agent in order to fulfill the request.

From https://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html

300 Multiple Choices

This provides certain choices to proceed with the request. Learn More

301 Moved Permanently

The resource is no longer residing at the current URI. Learn More

302 Found

The resource is temporarily residing at the different URI. Learn More

303 See Other

The response to the request can be located from different URI with the use of GET method. Learn More

304 Not Modified

This indicates that the resource has not been modified since the last request. Learn More

305 Use Proxy

The resource requested must accessed through the use of proxy. Learn More

306 (Unused)

This status code is no longer used but was used in the previous version and currently reserved. Learn More

307 Temporary Redirect

The resource requested is temporarily residing at the different URI. Learn More

308 Permanent Redirect (experimental)

The request must be done with another URI. Learn More

Client Error (4xx)

The class of status code is intended for cases in which the client seems to have erred.

From https://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html

400 Bad Request

The request cannot be completed due to bad syntax. Learn More

401 Unauthorized

The request requires user authentication. Learn More

402 Payment Required

This status code is reserved for future use and intended for request that needs payment of digital cash. Learn More

403 Forbidden

The server is refusing to respond with the request. Learn More

404 Not Found

The requested resource cannot be found by the server. Learn More

405 Method Not Allowed

The request method is not supported by the resource. Learn More

406 Not Acceptable

The resource requested is only capable of generating response which is not acceptable according to the accept headers sent in the request. Learn More

407 Proxy Authentication Required

To proceed, it is required that the client must first authenticate itself with the proxy. Learn More

408 Request Timeout

The client did not make any request within the time frame given by the server to wait. Learn More

409 Conflict

Whatever request cannot be done becauese of conflict. Learn More

410 Gone

Resource is no longer available. Learn More

411 Length Required

Server expecting content length. Learn More

412 Precondition Failed

Condition has failed, can not proceed to the request. Learn More

413 Request Entity Too Large

Request Entity is too large to be handled by the server. Learn More

414 Request-URI Too Long

Request-URI is too long to be handled by the server. Learn More

415 Unsupported Media Type

The server if refusing the request because the format is not supported. Learn More

416 Requested Range Not Satisfiable

The request cannot be supplied by the server. Learn More

417 Expectation Failed

The server cannot meet the requirements of the Expect request-header field. Learn More

418 I'm a teapot (RFC 2324)

This code is one of traditional IETF April Fools joke created in 1998. Learn More

420 Enhance Your Calm (Twitter)

This code is returned by the Twitter Search and Trends API to client when being rate limited.

422 Unprocessable Entity (WebDAV)

This indicates that the server was able to understand the content type of the request but unable to to process the contained instructions. Learn More

423 Locked (WebDAV)

This indicates that the resource requested is locked. Learn More

424 Failed Dependency (WebDAV)

This indicates that the current request failed because of the previous failure of request. Learn More

425 Reserved for WebDAV

This status code is reserved for the "WebDAV Advanced Collections Protocol".

426 Upgrade Required

This allows a server to state the precise protocol extensions, a given resource must be served with. Learn More

428 Precondition Required

This indicates that the origin server requires the request to be conditional. Learn More

429 Too Many Requests

This indicates that the client has sent too many requests in a given amount of time. Learn More

431 Request Header Fields Too Large

This indicates that the server do not accept the request due to large header fields. Learn More

444 No Response (Nginx)

The server returns no information to the client and closes the connection.

449 Retry With (Microsoft)

This status code is a Microsoft extension which indicates that the request should be retried after performing the appropriate action.

450 Blocked by Windows Parental Controls (Microsoft)

This status code is a Microsoft extension which is given when Windows Parental Controls are turned on and are blocking access to the given webpage.

451 Unavailable For Legal Reasons

This is used when resource access is denied for legal reasons. Learn More

499 Client Closed Request (Nginx)

This status code is a Nginx HTTP server extension which is used to log the case when the connection is closed by client while HTTP server is processing its request, making server unable to send the HTTP header back.

Server Error (5xx)

This class of status codes indicate cases in which the server is aware that it has erred or is incapable of performing the request.

From https://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html

500 Internal Server Error

The server encountered an unexpected condition preventing it to respond on the request. Learn More

501 Not Implemented

The server does not recognize the request method. Learn More

502 Bad Gateway

This indicates that the server while acting as a gateway or proxy, received an invalid response from the upstream server.

Learn More
503 Service Unavailable

The server is under maintenance or overloaded resulting to unable to respond to request. Learn More

504 Gateway Timeout

This indicates that the server while acting as a gateway or proxy, did not receive a timely response from the upstream server. Learn More

505 HTTP Version Not Supported

The HTTP version used in the request is not supported by the server. Learn More

506 Variant Also Negotiates (Experimental)

This indicates that the server has an internal configuration error. Learn More

507 Insufficient Storage (WebDAV)

This indicates that the method could not be performed on the resource because the server cannot store the representation needed to successfully complete the request. Learn More

508 Loop Detected (WebDAV)

This indicates that the server canceled an operation because it encountered an infinite loop while processing a request. Learn More

509 Bandwidth Limit Exceeded (Apache)

This status code is used by many servers but not specified in any RFCs.

510 Not Extended

This indicates that a further extensions to the request are required for the server to complete it. Learn More

511 Network Authentication Required

This indicates that the client needs to authenticate to gain network access. Learn More

598 Network read timeout error

This status code is not specified in any RFCs but is used by some HTTP proxies to signal a network read timeout.

599 Network connect timeout error

This status code is not specified in any RFCs but is used by some HTTP proxies to signal a network connect timeout.
